Detailed Description

A specific SLAdapter implementation for RESTful servers.

In addition to implementing the SLAdapter interface, SLRESTAdapter contains a single SLRESTContract to map remote methods to custom HTTP routes. This is only required if the HTTP settings have been customized on the server. When in doubt, try without.
